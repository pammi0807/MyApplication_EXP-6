# MyApplication_EXP-6

Aim

To develop an Android application that demonstrates the working of different layouts such as Linear Layout, Relative Layout, and Table Layout using XML and Java in Android Studio.

Algorithm
Start Android Studio and create a new Android project.
Design the main activity with buttons for Linear Layout, Relative Layout, and Table Layout.
Create separate activities for each layout type.
Design the Linear Layout using TextView, EditText, and Button components.
Design the Relative Layout using TextView and Buttons with relative positioning.
Design the Table Layout using TableLayout and TableRow elements.
Write Java code to navigate between activities using Intent.
Declare all activities in the AndroidManifest.xml file.
Run the application on an emulator or Android device.
Verify the output of all layouts successfully.
